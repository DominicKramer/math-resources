If $A$ is a matrix defined in {\sl Mathematica}, then \computerfont{Inverse[A]} will return the inverse of $A$, should it exist.  In the case where $A$ does not have an inverse {\sl Mathematica} will tell you the matrix is singular (see \acronymref{theorem}{NI}).
