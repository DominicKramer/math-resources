If $a$ is a matrix defined in {\sl SAGE}, then \computerfont{a.inverse()} will return the inverse of $a$, should it exist.  In the case where $a$ does not have an inverse {\sl SAGE} will tell you the matrix must be nonsingular (see \acronymref{theorem}{NI}).
