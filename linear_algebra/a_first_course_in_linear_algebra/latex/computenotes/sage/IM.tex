{\sl SAGE} will construct an identity matrix with the command \computerfont{identity\_matrix()}.  For example,
%
\begin{align*}
\computerfont{identity\_matrix(QQ, 3)}
\end{align*}
%
yields
%
\begin{align*}
\begin{bmatrix}
1 & 0 & 0 \\
0 & 1 & 0 \\
0 & 0 & 1
\end{bmatrix}
\end{align*}
