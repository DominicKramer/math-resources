The $2\times 2$ matrix
%
\begin{equation*}
\begin{bmatrix}
1&1\\
-1&-1
\end{bmatrix}
\end{equation*}
%
has $\csp{A}=\nsp{A}=\spn{\set{\colvector{1\\-1}}}$.
