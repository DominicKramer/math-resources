There is only one value of $k$ that will make this matrix have a zero determinant.
%
\begin{align*}
\detname{A} &= \begin{vmatrix} 2 & 4 \\ 3 & k \end{vmatrix} = 2k - 12
\end{align*}
%
so $\detname{A} = 0$ only when $k = 6$.
