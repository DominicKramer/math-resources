One of the fundamental identities of trigonometry is $\sin^2(x) + \cos^2(x) = 1$.  Thus, we have a dependence relation $2(\sin^2{x}) + 2(\cos^2{x}) + (-1)(2) = 0$, and the set is linearly dependent.
