Since $\colvector{4\\3}= \colvector{2\\1} + 2\colvector{1\\1}$, we have 
\begin{align*}
T\left(\colvector{4\\3}\right) 
&= \lt{T}{ \colvector{2\\1} + 2\colvector{1\\1}}
= \lt{T}{\colvector{2\\1}} + 2\,\lt{T}{\colvector{1\\1}}\\
%
&= \colvector{3\\4} + 2\colvector{-1\\2}
&= \colvector{1\\8}.
\end{align*}
