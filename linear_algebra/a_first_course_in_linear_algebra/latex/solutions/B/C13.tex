If $p(0) = 0$, then $a + b(0) + c(0^2) = 0$, so $a = 0$.  
Thus, we can write $Q = \setparts{p(x) = bx + cx^2}{b, c\in\complexes}$.
A linearly independent set that spans $Q$ is $B=\set{x, x^2}$, and this set forms a basis of $Q$.
