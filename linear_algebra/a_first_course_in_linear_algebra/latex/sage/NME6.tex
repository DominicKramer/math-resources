Recycling the nonsingular matrix from \acronymref{sage}{NME5} we can use Sage to verify the two new equivalences of \acronymref{theorem}{NME6}.
%
\begin{sageexample}
sage: A = matrix(QQ, [[ 2,  3, -3,  2,  8, -4],
...                   [ 3,  4, -4,  4,  8,  1],
...                   [-2, -2,  3, -3, -2, -7],
...                   [ 0,  1, -1,  2,  3,  4],
...                   [ 2,  1,  0,  1, -4,  4],
...                   [ 1,  2, -2,  1,  7, -5]])
sage: not A.is_singular()
True
sage: A.rank() == A.ncols()
True
sage: A.right_nullity() == 0
True
\end{sageexample}
