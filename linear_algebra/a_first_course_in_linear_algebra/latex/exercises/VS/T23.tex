Suppose $V$ is a vector space, $\vect{u}\neq\zerovector$ is a vector in $V$ and $\alpha,\,\beta\in\complex{\null}$. If $\alpha\vect{u}=\beta\vect{u}$, then $\alpha=\beta$.
