The following archetypes are either matrices or systems of equations with coefficient matrices.  For each matrix, compute the extended echelon form $N$ and identify the matrices $C$ and $L$.  Using \acronymref{theorem}{FS},  \acronymref{theorem}{BNS} and \acronymref{theorem}{BRS} express the null space, the row space, the column space and left null space of each coefficient matrix as a span of a linearly independent set.\newline\newline
