Suppose that $A$ is a square matrix.  Using the definition of reduced row-echelon form (\acronymref{definition}{RREF}) carefully, give a proof of the following equivalence:  Every column of $A$ is a pivot column if and only if $A$ is the identity matrix (\acronymref{definition}{IM}).
