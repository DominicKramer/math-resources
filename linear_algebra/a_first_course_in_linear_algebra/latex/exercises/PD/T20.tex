Suppose that $A$ is an $m\times n$ matrix and $\vect{b}\in\complex{m}$.  Prove that the linear system $\linearsystem{A}{\vect{b}}$ is consistent if and only if $\rank{A}=\rank{\augmented{A}{\vect{b}}}$.
