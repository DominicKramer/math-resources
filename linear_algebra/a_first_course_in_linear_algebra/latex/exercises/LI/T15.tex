Suppose that $\set{\vectorlist{v}{n}}$ is a set of vectors.   Prove that 
%
\begin{align*}
\set{\vect{v}_1-\vect{v}_2,\,\vect{v}_2-\vect{v}_3,\,\vect{v}_3-\vect{v}_4,\,\dots,\,\vect{v}_n-\vect{v}_1}
\end{align*}
%
 is a linearly dependent set.
