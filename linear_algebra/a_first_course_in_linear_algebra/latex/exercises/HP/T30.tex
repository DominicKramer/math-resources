Let $A$ be a square matrix of size $n$ with singular values $\scalarlist{\sigma}{n}$. Let $D$ be a diagonal matrix from the singular value decomposition of $A$, $A = UD \adjoint{V}$ (\acronymref{theorem}{SVD}).  Define the vector $\vect{d}$ by $\vectorentry{\vect{d}}{i}=\matrixentry{D}{ii}=\sigma_i$, $1\leq i\leq n$. Prove the following equality,
%
\begin{align*}
\matrixentry{A}{ii}
&=
\vectorentry{(\hadamard{U}{\conjugate{V}})\vect{d}}{i}
\end{align*}
