Working within the vector space $\complex{3}$, determine if 
$\vect{b} = \colvector{4\\3\\1}$ is in the subspace $W$,
\begin{equation*}
W =
\spn{\set{
\colvector{3\\2\\3}, 
\colvector{1\\0\\3}, 
\colvector{1\\1\\0}, 
\colvector{2\\1\\3}
}}
\end{equation*}
