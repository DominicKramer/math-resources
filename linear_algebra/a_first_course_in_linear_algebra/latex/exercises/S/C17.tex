Working within the vector space $\complex{4}$, determine if 
$\vect{b} = \colvector{2\\1\\2\\1}$ is in the subspace $W$,
\begin{equation*}
W = \spn{\set{
\colvector{1\\2\\0\\2}, 
\colvector{1\\0\\3\\1}, 
\colvector{0\\1\\0\\2}, 
\colvector{1\\1\\2\\0}
}}
\end{equation*}
