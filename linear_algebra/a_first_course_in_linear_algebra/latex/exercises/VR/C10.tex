In the vector space $\complex{3}$, compute the vector representation $\vectrep{B}{\vect{v}}$ for the basis $B$ and vector $\vect{v}$ below.
%
\begin{align*}
B&=\set{
\colvector{2\\-2\\2},\,
\colvector{1\\3\\1},\,
\colvector{3\\5\\2}
}
&
\vect{v}&=\colvector{11\\5\\8}
\end{align*}
