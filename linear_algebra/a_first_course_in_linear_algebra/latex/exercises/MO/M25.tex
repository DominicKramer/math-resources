Define a subset of $M_{3,3}$ by
%
\begin{equation*}
U_{33}=\setparts{
A\in M_{3,3}
}{
\matrixentry{A}{ij}=0\text{ whenever }i>j
}
\end{equation*}
%
Find a set $R$ so that $R$ is linearly independent and $\spn{R}=U_{33}$.
