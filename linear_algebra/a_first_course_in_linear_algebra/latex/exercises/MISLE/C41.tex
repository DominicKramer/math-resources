Use the inverse of a matrix to find all the solutions to the following system of equations.  %
\begin{align*}
x_1 + 2 x_2 - x_3 &= -3\\
2 x_1 + 5 x_2 - x_3 &= -4\\
-x_1 - 4 x_2 &= 2
\end{align*}
